The following abbreviations and acronyms appear in this guide or in the application:

* **2D** - 2-dimensional
* **3D** - 3-dimensional
* **AC** - Alternating current
* **EPU** - Embedded processing unit
* **PiP** - Picture-in-picture layout
* **USB** - Universal serial bus
